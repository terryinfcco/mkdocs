### Emmet Shortcuts

* html - create html boilerplate
* h1..h6 - heading h1 is the largest
* p - paragraph
* hr - horizontal rule
* ol - ordered list
* ul - unordered list
* li - list item
* img - image tag
* a - anchor tag
* table - tables
* tr - table row
* thead - table header
* tf - table footer
* form - form
